# Todos-List
A simple todos list application built with React. Allows users to add, delete, edit and update tasks.

#Features
Add tasks  
Delete tasks  
Edit tasks
Update tasks


#Tech Used
React, JavaScript, CSS, BOOTSTRAP, localstorage
